# Real-Estate-Secure Monorepo

This repository is organized as a single monorepo. The `main` branch is intended to contain the actual project folders.

## Projects

- `Assignment 1/`
- `Assignment 2/`
- `Assignment 3/`
- `Assignment 4/`
- `Assignment 5/`
- `gradecalculator/`
- `Real-Estate-Secure-Project/`

## Branch Workflow

Use short-lived task branches created from `main`.

Examples:

- `feat/assignment-2-readme-update`
- `fix/gradecalculator-cli-output`
- `docs/root-readme-refresh`

## Review Workflow

1. Create a task branch from `main`.
2. Make focused changes.
3. Push the branch.
4. Open a pull request into `main`.
5. Let your reviewer review and merge it.
